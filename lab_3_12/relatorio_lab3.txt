Função main com as alterações correspondentes ao pedido no enunciado 

#include <stdio.h>
int main(void) {
  unsigned int x = 0x87654321;
  unsigned int y, z;

  /* o byte menos significativo de x e os outros bits em 0 */
  y = x & 0x000000FF;

  /* o byte mais significativo com todos os bits em '1' 
     e os outros bytes com o mesmo valor dos bytes de x */
  z = 0xFF000000|x;

  printf("%08x %08x\n", y, z);
}

é possível analisar que como y deve representar o valor com o byte menos significativo igual ao byte de x
e todos os outros igual a 0, ao realizar a operação "&" na comparação com o valor 0x000000FF, todos os valores estarão 
automaticamente 'desligados', assim comprindo com o requisito.
Assim como z, que precisa ter os bytes mais significativos igual a 1, enquanto o resto é igual aos valores de x,
usando o operador "|" como os primeiros 2 bits estão ligados em  0xFF000000 ele sempre terá o valor ff no inicio, deixando 
os outros valores a dizer do valor de x (já que seus outros bites são 0)

assim o valor do output é 
00000021 ff654321
